Title: A Comprehensive Spell Checker
Description: This is a comprehensive Spell Checker application which uses an internal database of over 450,000 words! Uses the Soundex and Levenshtein Distance Algorithms to fully emulate a word processor spell checker. The number of possible results can be adjusted too. All algorithms, additional work etc is fully documented. Votes welcome.
